# Javascript Comparison Operators

| Operator      |   Meaning                  |
| ------------- |:--------------------------:| 
| ==            | equality                   |
| ===           | strict-equality            |
| !=            | not equal to               |
| <=            | less than or equal to      |
| >=            | greater than or equal to   |
| <             | less than                  |
| >             | greater than               |
